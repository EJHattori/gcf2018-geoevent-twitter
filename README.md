# gcf2018-geoevent-twitter
GISコミュニティフォーラム2018年GeoEventデモ用

* 概要
** SNSにより人、モノ、商品などに対する好感度をリアルタイムに測定して地図上に可視化します。これにより特定の商品やサービスなどのイメージをビジュアルに地図上で把握することが可能となります。2018年5月末に開催されましたGISコミュニティフォーラムにてデモ展示しました内容を一部公開します。
